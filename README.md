# Enterprise CI/CD DevOps Pipeline on AWS

## Project Overview
This project implements a fully automated CI/CD pipeline for deploying a three-tier web application on AWS using modern DevOps tools and practices.

## Student Details
- **Name:** Rishav
- **Project Title:** Enterprise CI/CD DevOps Pipeline on AWS

## Problem Statement
Traditional web application deployment is manual, time-consuming, and error-prone. This project solves these challenges by automating the entire deployment process.

## Tools & Technologies Used
- **Frontend:** React.js
- **Backend:** Node.js
- **Database:** MongoDB
- **Containerization:** Docker
- **Container Orchestration:** Kubernetes (AWS EKS)
- **Infrastructure as Code:** Terraform
- **CI/CD:** Jenkins
- **GitOps:** ArgoCD
- **Code Quality:** SonarQube
- **Monitoring:** Prometheus & Grafana
- **Cloud:** AWS (EC2, EKS, ECR, IAM)
- **Version Control:** GitHub

## Architecture
Three-tier architecture with automated CI/CD pipeline on AWS EKS.
